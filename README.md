```text
        DIO -|1      24|- GRD1

        STB -|3      22|- GND
        VDD -|4      21|- GRD3
       SEG1 -|5      20|- GRD4
       SEG2 -|6      19|- GND
       SEG3 -|7      18|- SEG14/GRD5
       SEG4 -|8      17|- SEG13/GRD6
       SEG5 -|9      16|- SEG12/GRD7
       SEG6 -|10     15|- SEG11

             +---------+
```
